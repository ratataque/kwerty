## Overview

Kitty is a GPU-accelerated terminal emulator focused on performance and features. Both ratataque and GrimalDev use it as their primary terminal, configured with Catppuccin Mocha and JetBrains Mono.

## Key Configuration

```conf
# Font
font_family      JetBrains Mono
bold_font        JetBrains Mono Bold
italic_font      JetBrains Mono Italic
font_size        12.0

# Window
window_padding_width  8
hide_window_decorations yes
background_opacity    0.95

# Performance
repaint_delay    6
input_delay      1
sync_to_monitor  yes
```

## Keybindings

Kitty uses `ctrl+shift` as the default modifier for its own keybindings. Custom bindings are added for tmux integration:

```conf
# Tabs
map ctrl+shift+t new_tab
map ctrl+shift+right next_tab
map ctrl+shift+left previous_tab

# Scrollback
map ctrl+shift+k scroll_line_up
map ctrl+shift+j scroll_line_down
map ctrl+shift+u scroll_page_up
map ctrl+shift+d scroll_page_down
```

## Platform Notes

On Linux, kitty integrates with Hyprland window rules for per-workspace opacity. On macOS, the `hide_window_decorations` setting removes the native title bar while keeping window management via yabai.
