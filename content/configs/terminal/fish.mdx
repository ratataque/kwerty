## Overview

Fish is the shell of choice for both contributors. Unlike bash/zsh, Fish works well out of the box with autosuggestions, syntax highlighting, and tab completions without plugins.

## Core Config

```fish
# config.fish
set -g fish_greeting ""
set -gx EDITOR nvim
set -gx VISUAL nvim

# Path
fish_add_path ~/.local/bin
fish_add_path ~/.cargo/bin

# Aliases
alias vim="nvim"
alias ls="eza --icons"
alias ll="eza -la --icons"
alias tree="eza --tree --icons"
alias cat="bat"
alias g="git"
alias lg="lazygit"
```

## Abbreviations

Fish abbreviations expand on space, unlike aliases which are literal replacements:

```fish
abbr -a gc "git commit"
abbr -a gp "git push"
abbr -a gpl "git pull"
abbr -a gco "git checkout"
abbr -a gs "git status"
abbr -a gd "git diff"
abbr -a ga "git add"
abbr -a gb "git branch"
```

## Functions

Custom Fish functions live in `~/.config/fish/functions/`:

```fish
# mkcd.fish - create and enter directory
function mkcd
    mkdir -p $argv[1] && cd $argv[1]
end
```

## Integration with tmux

Fish is configured to attach to or create a tmux session on terminal startup (except in nested sessions):

```fish
if status is-interactive
    and not set -q TMUX
    tmux attach -t default 2>/dev/null; or tmux new-session -s default
end
```

## Prompt

Both contributors use Starship prompt (see starship config) rather than Fish's built-in prompt.
