## Overview

skhd is the hotkey daemon that provides keyboard bindings for yabai window management on macOS. The bindings mirror ratataque's Hyprland setup as closely as possible.

## Keybindings

```bash
# skhdrc

# Focus windows (alt + hjkl)
alt - h : yabai -m window --focus west
alt - j : yabai -m window --focus south
alt - k : yabai -m window --focus north
alt - l : yabai -m window --focus east

# Move windows (shift + alt + hjkl)
shift + alt - h : yabai -m window --swap west
shift + alt - j : yabai -m window --swap south
shift + alt - k : yabai -m window --swap north
shift + alt - l : yabai -m window --swap east

# Resize windows
shift + alt - left  : yabai -m window --resize left:-20:0
shift + alt - down  : yabai -m window --resize bottom:0:20
shift + alt - up    : yabai -m window --resize top:0:-20
shift + alt - right : yabai -m window --resize right:20:0

# Spaces
alt - 1 : yabai -m space --focus 1
alt - 2 : yabai -m space --focus 2
alt - 3 : yabai -m space --focus 3
alt - 4 : yabai -m space --focus 4
alt - 5 : yabai -m space --focus 5

# Move to space
shift + alt - 1 : yabai -m window --space 1
shift + alt - 2 : yabai -m window --space 2
shift + alt - 3 : yabai -m window --space 3
shift + alt - 4 : yabai -m window --space 4
shift + alt - 5 : yabai -m window --space 5

# Layout
alt - f : yabai -m window --toggle zoom-fullscreen
alt - t : yabai -m window --toggle float
alt - r : yabai -m space --rotate 270

# Applications
alt - return : open -a kitty
```

## Philosophy

The keybindings intentionally use `alt` as the modifier instead of `cmd` to avoid conflicts with macOS system shortcuts. This matches the `SUPER` modifier approach used in Hyprland.
