## Overview

yabai is GrimalDev's tiling window manager for macOS. It provides BSP (binary space partitioning) tiling, space management, and window manipulation. Paired with skhd for keybindings and karabiner for key remapping.

## Configuration

```bash
#!/usr/bin/env sh
# yabairc

# Layout
yabai -m config layout bsp
yabai -m config window_placement second_child
yabai -m config split_ratio 0.50

# Gaps
yabai -m config top_padding    8
yabai -m config bottom_padding 8
yabai -m config left_padding   8
yabai -m config right_padding  8
yabai -m config window_gap     8

# Mouse
yabai -m config mouse_follows_focus on
yabai -m config mouse_modifier alt
yabai -m config mouse_action1 move
yabai -m config mouse_action2 resize

# Window appearance
yabai -m config window_opacity on
yabai -m config active_window_opacity 1.0
yabai -m config normal_window_opacity 0.95
```

## Application Rules

```bash
# Float system apps
yabai -m rule --add app="^System Preferences$" manage=off
yabai -m rule --add app="^Calculator$" manage=off
yabai -m rule --add app="^Karabiner" manage=off
yabai -m rule --add app="^Archive Utility$" manage=off

# Assign to spaces
yabai -m rule --add app="^Firefox$" space=2
yabai -m rule --add app="^Slack$" space=3
```

## Integration

yabai works with skhd (see skhd config) for keyboard shortcuts. All window operations are performed through skhd bindings, providing a Hyprland-like experience on macOS.
