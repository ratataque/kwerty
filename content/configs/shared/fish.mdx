## Overview

Fish is the shell of choice for both contributors. Unlike bash/zsh, Fish works well out of the box with autosuggestions, syntax highlighting, and tab completions without plugins.

## Core Config

```fish
# config.fish
set -g fish_greeting ""
set -gx XDG_CONFIG_HOME $HOME/.config
set -gx EDITOR nvim
set -gx VISUAL nvim

# Vi key bindings
fish_vi_key_bindings

# Path
fish_add_path /opt/homebrew/bin
fish_add_path /opt/homebrew/sbin
fish_add_path $HOME/.local/bin
fish_add_path $HOME/.cargo/bin
fish_add_path $HOME/go/bin
fish_add_path $HOME/.bun/bin
```

## Functions

Custom Fish functions for common operations:

```fish
# sudo.fish - re-run last command with sudo
function sudo
    if test (count $argv) -eq 0
        command sudo $history[1]
    else
        command sudo $argv
    end
end

# brew.fish - update, upgrade, cleanup
function brew
    if test (count $argv) -eq 0
        command brew update && command brew upgrade && command brew cleanup
    else
        command brew $argv
    end
end

# jqf.fish - interactive jq with fzf
function jqf
    echo '' | fzf --print-query --preview "cat $argv[1] | jq {q}"
end

# share.fish - create temp share via transfer.sh
function share
    curl --upload-file $argv[1] "https://transfer.sh/$argv[1]"
end

# mfzf.fish - man page fuzzy finder
function mfzf
    man -k . | fzf --prompt='Man> ' | awk '{print $1}' | xargs -r man
end
```

## Shell Integrations

# fzf key bindings
